  1. What is DOM and what is the purpose of DOM?
    Ans: DOM stands for Document Object Model. DOM is a programming interface of a html and xml document. By using DOM we can access the element , ineer text, tag etc of a html document and also can change the structure of document for interactive design and better user experience.
  2. How will you select HTML elements using DOM? Name the DOM methods.
    Ans: To get the element of an id using document.getElementById(), To get the element of a class using document.getElementsByClassName, To get the element of a html tag using document.getElementsByTagName.
  3. What is event bubble?
    Ans: Event bubble is the concept of DOM event. It is the concept that a event first applied to the innermost target element and triggers graduatly the parent element of the node.
  4. What is a callback function and why will you use it?
    Ans: When a function call another function that is called a callback function.
